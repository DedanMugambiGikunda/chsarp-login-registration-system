# chsarp-login-registration-system
A secure Windows Forms Login and Registration application built with C# and ADO.NET featuring parametized SQL queries to safeguard authentication.

Features
- User Registration: Validates empty fields, ensures password confirmation matches, and checks for existing usernames to prevent duplicates.
- Security First Architecture: Utilizes parameterized SQL commands across all database transactions to completely mitigate SQL Injection vulnerabilities.
- Automatic ID Generation: Designed to interface seamlessly with structured relational database

Language: C#
- Framework: .NET Framework / Windows Forms (WinForms)
- Database: Microsoft SQL Server (LocalDB)
- Data Access: ADO.NET (SqlClient)
